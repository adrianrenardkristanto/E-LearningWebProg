HTML Dasar

Struktur dasar HTML dan elemen penting

By E-Learning Team / April 18, 2025

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML berfungsi sebagai kerangka dasar dari semua situs web, yang memungkinkan browser memahami dan menampilkan konten dengan struktur yang rapi.

1. Struktur Dasar HTML

Sebuah dokumen HTML biasanya dimulai dengan struktur berikut:

    <!DOCTYPE html>
    <html lang="id">
      <head>
        <meta charset="UTF-8">
        <title>Judul Halaman</title>
      </head>
      <body>
        <h1>Halo, dunia!</h1>
        <p>Ini adalah paragraf pertama saya.</p>
      </body>
    </html>

2. Elemen Penting dalam HTML

  • <h1> sampai <h6> - Menandai heading atau judul.
  • <p> - Membuat paragraf.
  • <a> - Membuat tautan (link).
  • <img> - Menampilkan gambar.
  • <ul>, <ol>, <li> - Membuat daftar.
  • <div> - Elemen kontainer untuk layout.
  • <span> - Elemen inline untuk teks.

3. Penulisan Tag dan Atribut

Setiap tag HTML dapat memiliki atribut, yang memberikan informasi tambahan. Contohnya:

    <a href="https://www.example.com">Kunjungi situs</a>

HTML sangat penting dalam dunia web, dan memahami struktur serta elemen-elemennya adalah langkah awal untuk menjadi web developer yang handal.

Tips: Latihan membuat struktur halaman sederhana menggunakan HTML adalah cara terbaik untuk memahami dan menguasai konsep ini.
