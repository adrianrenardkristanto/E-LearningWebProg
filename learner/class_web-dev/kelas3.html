CSS untuk Styling

Menggunakan CSS untuk desain web

By E-Learning Team / April 18, 2025

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak elemen HTML di halaman web. Dengan CSS, kita dapat menambahkan warna, mengatur ukuran font, mengatur jarak antar elemen, dan membuat desain web menjadi lebih menarik serta responsif.

1. Cara Menyisipkan CSS

CSS dapat digunakan dengan tiga cara:

  • Inline: langsung di dalam tag HTML menggunakan atribut style
  • Internal: ditulis dalam tag <style> di dalam <head>
  • Eksternal: ditulis dalam file .css terpisah dan disambungkan menggunakan <link>

2. Contoh Dasar CSS

        /* File: styles.css */
        body {
          font-family: Arial, sans-serif;
          background-color: #f0f0f0;
        }
        
        h1 {
          color: #3f37c9;
          text-align: center;
        }
        
        p {
          font-size: 1rem;
          line-height: 1.6;
        }

3. Selektor dan Properti

CSS menggunakan selektor untuk memilih elemen HTML yang ingin diubah, dan properti untuk menentukan gaya apa yang akan diterapkan. Misalnya:

        p {
          color: #333;
          margin-bottom: 10px;
        }

4. Responsif dan Media Query

Dengan @media query, kita bisa membuat tampilan yang menyesuaikan dengan ukuran layar pengguna. Ini penting untuk memastikan website tampil baik di berbagai perangkat.

        @media (max-width: 600px) {
          body {
            font-size: 14px;
          }
        }

Kesimpulan: CSS sangat penting dalam pengembangan web modern karena memungkinkan pengembang untuk menciptakan pengalaman visual yang menarik, konsisten, dan profesional. Dengan menguasai CSS, kamu bisa mengubah halaman HTML biasa menjadi website yang estetis dan user-friendly.
